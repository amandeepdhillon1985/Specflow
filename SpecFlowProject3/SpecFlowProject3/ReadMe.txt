Language used is C#
Testing framework used is Specflow and MSTest 
Testing tool used is selenium 
Go to Test and open the Test explorer to run/debug the tests
The feature folder contain the features written in gherkin language 
StepDefinitions folder contain all the bindings or the code behing the gherkin steps
Hooks folder contain the initializer class that has tags and before and after test scenarios
PageMap folder contain all the pages with elements and if any element is removed or changed,
it would require change at only page map level for fixing
